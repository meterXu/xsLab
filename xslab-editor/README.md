# xslab-editor
> 可视化图表编辑器前端，支持ECharts和表格等元素可视化配置

## 快速开始

``` bash
# 克隆
git clone https://github.com/meterXu/xsLab.git

# 初始化
cd xslab-editor
npm install

# 运行
npm run dev

# 编译
npm run build
```
